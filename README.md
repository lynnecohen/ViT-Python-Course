**Highlighted Projects**

**Rick and Morty API Data Export**

  **Overview:**
  A Python script that retrieves paginated data from the Rick and Morty public API and exports complete datasets (characters, episodes, locations) to an Excel workbook using openpyxl.
  
  **What it does:**
  * Fetches API data and handles pagination via recursive traversal
  * Dynamically builds worksheet headers
  * Writes full datasets into separate Excel worksheets
  * Demonstrates the ability to work with real-world API patterns and produce structured spreadsheet outputs.
